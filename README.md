## Installation <a id="installation"></a>

To use the Nav Bar Module in your project, you can clone the repository:

```bash
git clone https://github.com/rhamseyswork/Nav-Bar.git
```

To use the Nav Bar Module in your Terminal:

```bash
npm run start
```
